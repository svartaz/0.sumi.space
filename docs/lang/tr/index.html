# turkish

  • consonant
    • ç, c ↦ ṭ, ḍ
    • ş, j ↦ ṣ, ẓ
    • y ↦ j
    • h ↦ x
  • vowel
    • ı, i ↦ î, ǐ
    • a, e ↦ â, ǎ
    • o, ö ↦ ô, ǒ
    • u, ü ↦ û, ǔ
  • the same diacritics (ˆ or ˇ) in a row on vowels in a word can be omitted except the initial one.
    • öğrenci ↦ ǒğrǎncǐ ↦ ǒğranḍi
  • the initial vowel with no diacritic in a word is thought to be with ˆ by default.
    • arkadaş ↦ ârkâdâṣ ↦ ârkadaṣ ↦ arkadaṣ
